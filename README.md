# City Branding Multimodal Lampung Selatan


> **Mengungkap Citra Kawasan Pariwisata Berdasarkan Data Sosial Media dengan Deep Learning Multimodal untuk Peningkatan City Branding dan Ekonomi Lokal**

Proyek ini berfokus pada analisis citra pariwisata Kabupaten Lampung Selatan berdasarkan data media sosial Instagram. Data dikumpulkan melalui proses crawling menggunakan sejumlah hashtag pariwisata, kemudian dianalisis secara multimodal dengan memanfaatkan informasi visual dari gambar dan informasi tekstual dari caption atau metadata. Hasil analisis diarahkan untuk memahami bagaimana Lampung Selatan direpresentasikan sebagai destinasi wisata, serta bagaimana temuan tersebut dapat mendukung strategi city branding dan penguatan ekonomi lokal.

## Latar Belakang

Media sosial menjadi ruang penting dalam pembentukan persepsi publik terhadap suatu daerah wisata. Unggahan wisatawan, komunitas lokal, pelaku usaha, dan akun promosi daerah dapat menggambarkan citra kawasan secara organik. Dengan pendekatan deep learning multimodal, proyek ini berupaya membaca pola visual dan tekstual dari konten Instagram untuk mengidentifikasi elemen-elemen utama city branding pariwisata Lampung Selatan.

Analisis dilakukan menggunakan kerangka **4A pariwisata**, yaitu:

| Komponen | Fokus Analisis |
| --- | --- |
| **Attraction** | Daya tarik wisata seperti pantai, gunung, pulau, budaya, event, spot foto, dan pengalaman wisata. |
| **Accessibility** | Akses menuju destinasi, kondisi jalan, transportasi, rute, konektivitas, dan kemudahan mobilitas wisatawan. |
| **Amenities** | Fasilitas pendukung wisata seperti penginapan, kuliner, area parkir, toilet, tempat ibadah, pusat informasi, dan fasilitas publik lainnya. |
| **Ancillary** | Dukungan kelembagaan dan layanan tambahan seperti pengelola destinasi, promosi wisata, komunitas, keamanan, kebijakan, dan layanan pendukung pariwisata. |
